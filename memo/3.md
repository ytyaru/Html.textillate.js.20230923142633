# 台本書式

## メッセージウインドウ型

改行コード系
```
二行以上でメッセージ窓の改ページをする

一窓内で二行以上の改行するには
\n
のようにエスケープシーケンスを使う。
\n\n
のように連続できるし\nこのように\n\n行中にも書ける。

\\n
でエスケープシーケンス文字をそのまま表示する。
台本で改行しつつ表示では改行したくないときは\
行末にバックスペースをつける。
```

待機系
```
行末の黒下三角形は入力待機。▼
ここでユーザ入力されるまで次のメッセージ表示を待機する。

:texta.delay 3000
テキストの一字表示速度を設定する。設定値はこれ以降永続する。

:texta.delay.once 1000
次のメッセージウインドウで表示するテキストの一字表示速度を一回だけ指定の速度にする。

:texta.delay.piriod 1000
:texta.delay.comma 1000
句読点速度、。

:texta.delay.once 1000
一文字単位で{3000}ウェイト時間を指定する。

```
